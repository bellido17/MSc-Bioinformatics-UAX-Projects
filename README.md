\# MSc Bioinformatics Projects - UAX 🧬💻



Collection of academic projects and scripts developed during the Master's in Bioinformatics at Universidad Alfonso X El Sabio (UAX). 



\## 📂 Repository Structure



\### 1. Protein-Network-LinkedList

\* \*\*Language:\*\* Python

\* \*\*Description:\*\* Object-Oriented Programming (OOP) implementation of a dynamic Linked List from scratch to model Protein-Protein Interactions (PPI). The algorithm successfully builds network nodes, simulates connectivity, and identifies biological "Hub" proteins (e.g., P53). Focus on low-level memory management and Big O complexity.



\### 2. RNA-Protein-Translator

\* \*\*Language:\*\* Python (Jupyter Notebook)

\* \*\*Description:\*\* A modular parsing tool that ingests standard `.fasta` files, cleans mRNA sequences, and translates them into amino acid chains using a customized genetic code dictionary algorithm. Includes binary file data handling and robust string manipulation.



### 3. GEO-Expression-Analysis-R
* **Language:** R (RMarkdown)
* **Description:** Comprehensive transcriptomic data analysis pipeline using a real Gene Expression Omnibus (GEO) dataset (GSE152075). Demonstrates proficiency in bioinformatics R packages, reproducible research practices (generating HTML/PDF automated reports), and statistical evaluation of genomic expression data.



### 4. Multiomics-Data-Integration
* **Language:** Python (Jupyter Notebook)
* **Description:** Advanced integration and analysis of multi-layered biological data (Multiomics). Showcases the ability to handle complex datasets, apply computational data science techniques to extract cross-omics insights, and document findings in a comprehensive scientific report.



### 5. Variant-Annotation-VEP-Pipeline
* **Language:** Bash (Shell Scripting)
* **Description:** Automation of clinical genomics pipelines using Bash. Implemented the Ensembl Variant Effect Predictor (VEP) tool to functionally annotate patient exome data. Includes the generation of comprehensive summary reports identifying clinically relevant genetic variants, demonstrating strong command-line (Linux) proficiency.
