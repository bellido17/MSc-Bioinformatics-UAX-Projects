\# MSc Bioinformatics Projects - UAX 🧬💻



Collection of academic projects and scripts developed during the Master's in Bioinformatics at Universidad Alfonso X El Sabio (UAX). 



\## 📂 Repository Structure



\### 1. Protein-Network-LinkedList

\* \*\*Language:\*\* Python

\* \*\*Description:\*\* Object-Oriented Programming (OOP) implementation of a dynamic Linked List from scratch to model Protein-Protein Interactions (PPI). The algorithm successfully builds network nodes, simulates connectivity, and identifies biological "Hub" proteins (e.g., P53). Focus on low-level memory management and Big O complexity.



\### 2. RNA-Protein-Translator

\* \*\*Language:\*\* Python (Jupyter Notebook)

\* \*\*Description:\*\* A modular parsing tool that ingests standard `.fasta` files, cleans mRNA sequences, and translates them into amino acid chains using a customized genetic code dictionary algorithm. Includes binary file data handling and robust string manipulation.
